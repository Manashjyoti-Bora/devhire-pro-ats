# 🚀 DevHire Pro ATS — Next-Gen Tech Job Portal

A professional, modern Applicant Tracking System & Job Portal built specifically for tech recruiters and software engineers.

## ✨ Features
- ⚡ **Real-time Keyword & Skill Filtering:** Instant search across job titles, companies, and required tech stacks.
- 🎨 **Glassmorphic UI & Theme Toggle:** Seamless switching between Dark and Light mode aesthetics.
- 📊 **Interactive Application Management:** Live tracking of submitted job applications.

## 🛠️ Tech Stack
- **Frontend Framework:** React 19 + Vite
- **Icons & Styling:** Lucide React, Modern CSS Grid & Variables
